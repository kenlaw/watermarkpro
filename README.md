======================================== iClass WatermarkPro (版本 1.26020309)
這是一個基於瀏覽器的單文件 (Single-file) 浮水印添加工具，專為保護教學資源、機密文件與圖片設計。無需安裝伺服器，所有處理皆在本地端 (Client-side) 完成，確保資料安全。
[ 專案簡介 ] iClass WatermarkPro 是一個整合了 React、Tailwind CSS 與強大 PDF 處理庫 (pdf-lib, pdf.js) 的 HTML5 應用程式。只要下載一個 .html 檔案，即可在任何現代瀏覽器中運行。
它特別針對「防止 AI 去除浮水印」進行了優化，並支援將 PDF 文字轉為外框 (Rasterization)，防止內容被選取或複製。
[ 核心功能 ]
1. 批量處理系統 (Batch Processing)
   * 支援一次拖放上傳多個 PDF 或圖片文件 (PNG/JPG)。
   * 單一文件直接下載，多個文件自動打包為 ZIP 壓縮檔。
   * 支援 PDF 頁數上限提示更新至 500 頁。
2. 強效防護技術 (Anti-AI & Security)
   * 雜訊干擾 (Noise)：在浮水印上添加隨機噪點，干擾 AI 修復算法。
   * 隨機化 (Randomization)：微調每個浮水印的角度與位置，打破規律性。
   * 漸層填充 (Gradient)：使用非純色填充，增加去除難度。
   * 文字轉外框 / 內容光柵化 (Convert to Outlines / Rasterize) [Beta]：
      * 將 PDF 每一頁強制轉換為高解析度圖片。
      * 徹底鎖定文字內容，防止被選取、搜尋或複製。
      * 支援「超高清 (Ultra)」模式 (360 DPI)，確保轉圖後文字邊緣依然銳利。
3. 智慧畫質管理
   * 提供三種輸出畫質選項，平衡檔案大小與清晰度：
      * 超高清 (Ultra)：360 DPI，100% 品質，專為「文字轉外框」設計，效果最接近向量原檔。
      * 高畫質 (High)：216 DPI，95% 品質，適合一般印刷。
      * 普通 (Normal) / 快速 (Low)：適合螢幕瀏覽與快速傳輸。
4. 一鍵快速設定 (Quick Presets)
   * 一般保密：標準浮水印設定。
   * 離線課本：高密度、低透明度，專用版權文字。
   * 僅轉外框 (無水印)：隱藏浮水印，僅執行 PDF 光柵化，用於單純鎖定文件內容。
5. 真實預覽技術
   * 整合 pdf.js，直接渲染 PDF 第一頁內容進行預覽，所見即所得 (WYSIWYG)。
   * 支援多文件預覽切換。
[ 如何使用 ]
1. 下載本 Repo 中的 index.html 檔案。
2. 直接使用 Chrome、Edge 或 Safari 瀏覽器打開該檔案。
3. 拖放文件，調整設定，點擊下載即可。
[ 技術棧 ]
* 核心框架: React 18 (經由 Babel Standalone 編譯)
* 樣式: Tailwind CSS (CDN)
* PDF 處理: pdf-lib (修改與生成), pdf.js (預覽渲染)
* 壓縮: JSZip (批量下載打包)
* 架構: Single-file HTML (無後端，純前端運作)
